Function Application

Status: planned

Functions are applied prefix-style: the function name precedes its arguments. 

; Planned syntax:
sqrt 16         ; function application: 4
max 3 7         ; two-argument function: 7

The combination of prefix functions and postfix operators creates a natural flow: data flows left-to-right through transformations.
